# 剧本资产表生成 Skill

## 适用平台
本 Skill 为通用 AI 编程助手指令，适用于 Marvis、Cursor、GitHub Copilot、Claude Code、Codex CLI、Windsurf 等任意支持 Python 执行环境的 AI 助手。

## 使用方式
将本 Skill 文档内容提供给 AI 助手，然后发送剧本文件（PDF / Word / Excel 均可）+ 可选集数范围（如"第17集到最后"，默认全剧本），AI 助手按本文档流程执行。

---

## 一、读取与提取

1. 读取文件全文：PDF 用 `read_file`，Word（.docx）用 python-docx 解析，Excel（.xlsx）用 openpyxl 解析
2. 定位指定集数范围（如"第十七集"至末尾）
3. 逐集逐场提取三类资产，**不得遗漏任何角色/场景/道具**（包括画外音、闪回场景、仅提及未出场的角色）
4. 去重归类为三个清单：场景 / 人物 / 道具

---

## 二、Excel 生成（核心）—— openpyxl 精确实现

> **关键原则：用 openpyxl 重新生成整个 xlsx，不要用修改已有文件的方式，避免格式残留。**

### 2.1 工作簿结构

| Sheet 名 | 内容 |
|----------|------|
| 场景资产 | 剧本指定范围内所有场景 |
| 人物资产 | 所有人物（含画外音/声音出场/仅提及） |
| 道具资产 | 所有道具（含服装、设备、文件、武器等） |

### 2.2 列结构（三个 Sheet 统一，5 列）

| 列 | 列名 | 列宽 |
|----|------|------|
| A | 资产名 | ~20 |
| B | 完成状态 | ~14 |
| C | 出现集数/场次 | ~30 |
| D | 用途描述 | ~50 |
| E | 外貌描写 | ~50 |

### 2.3 表头样式（三 Sheet 统一）

```python
from openpyxl.styles import PatternFill, Font, Alignment

header_fill = PatternFill(start_color="003366", end_color="003366", fill_type="solid")
header_font = Font(name="微软雅黑", bold=True, color="FFFFFF", size=11)
header_alignment = Alignment(horizontal="center", vertical="center", wrap_text=True)
```

每个 Sheet 第一行写入表头后：
- 对 A1:E1 应用 `header_fill` + `header_font` + `header_alignment`
- 冻结首行：`ws.freeze_panes = "A2"`
- 行高设置为 30

### 2.4 数据写入

- 从第 2 行开始写入数据
- B列（完成状态）统一写入 "未完成"
- 字体统一：`Font(name="微软雅黑", size=10)`，颜色不设置（默认黑色）
- 对齐：垂直居中，自动换行

### 2.5 数据验证（下拉菜单）—— **关键步骤**

```python
from openpyxl.worksheet.datavalidation import DataValidation

dv = DataValidation(
    type="list",
    formula1='"未完成,已完成,正在完成,暂缓"',
    allow_blank=True
)
dv.error = "请选择有效状态"
dv.errorTitle = "无效输入"
ws.add_data_validation(dv)
dv.add(f"B2:B{last_row}")
```

### 2.6 条件格式 —— **最容易出错的部分，必须严格按以下方式实现**

> **核心规则：仅设置 PatternFill 填充色，绝不设置 Font。如果设置了 Font，下拉切换后字体会变，导致翻车。**

```python
from openpyxl.formatting.rule import CellIsRule
from openpyxl.styles import PatternFill

# 只定义 fill，不定义 font
red_fill    = PatternFill(start_color="FFC7CE", end_color="FFC7CE", fill_type="solid")
green_fill  = PatternFill(start_color="C6EFCE", end_color="C6EFCE", fill_type="solid")
yellow_fill = PatternFill(start_color="FFEB9C", end_color="FFEB9C", fill_type="solid")
gray_fill   = PatternFill(start_color="D9D9D9", end_color="D9D9D9", fill_type="solid")

range_str = f"B2:B{last_row}"

ws.conditional_formatting.add(range_str, CellIsRule(operator="equal", formula=['"未完成"'],   fill=red_fill))
ws.conditional_formatting.add(range_str, CellIsRule(operator="equal", formula=['"已完成"'],   fill=green_fill))
ws.conditional_formatting.add(range_str, CellIsRule(operator="equal", formula=['"正在完成"'], fill=yellow_fill))
ws.conditional_formatting.add(range_str, CellIsRule(operator="equal", formula=['"暂缓"'],     fill=gray_fill))
```

**禁止的做法（会导致颜色丢失或字体变色）：**
- ❌ 在 CellIsRule 中传入 `font=Font(...)` —— 这会覆盖默认黑色字体
- ❌ 直接逐单元格 set fill 而不加条件格式 —— 下拉切换后颜色不会变
- ❌ 使用 FormulaRule 或复杂公式 —— 容易出错，CellIsRule 最稳定
- ❌ 先设默认红色 fill 再加条件格式 —— 条件格式优先级高于直接 fill，可能导致颜色不显示
