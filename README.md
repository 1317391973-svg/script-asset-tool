# 剧本资产表生成 Skill

> 一键将剧本（PDF/Word/Excel）解析为结构化资产表，自动生成带状态追踪和颜色标记的 Excel。

[![Gitee stars](https://gitee.com/white-clad-er-zhuzi/script-asset-tool/badge/star.svg)](https://gitee.com/white-clad-er-zhuzi/script-asset-tool)

## 这是什么

短剧/网文编剧的资产管理工具。把一份剧本丢进去，自动输出一份 Excel，包含：

| 分表 | 内容 | 用途 |
|------|------|------|
| 场景资产 | 所有场景清单 | 拍摄统筹、美术置景 |
| 人物资产 | 所有角色（含画外音） | 选角、造型 |
| 道具资产 | 所有道具/服装/设备 | 道具组采购准备 |

每个资产自动标注**出现集数场次**、**用途描述**、**外貌描写**，并配有**颜色标记的下拉状态追踪**（未完成/正在完成/已完成/暂缓）。

## Excel 效果

| 资产名 | 完成状态 | 出现集数/场次 | 用途描述 | 外貌描写 |
|--------|----------|--------------|----------|----------|
| 中央医院 | 🔴 未完成 | 第17集 第1场 | 丽贝卡抢救场景 | 白色走廊，急救设备 |
| 丽贝卡 | 🟡 正在完成 | 第17-24集 | 女主角，悔悟赎罪 | 30岁，金发憔悴 |
| 遗嘱文件 | 🟢 已完成 | 第18集 第3场 | 维克多的秘密遗嘱 | 牛皮纸信封，火漆封印 |

状态列支持下拉切换，颜色自动跟随变化。

## 使用方法

将本 Skill 文档部署到 Marvis 桌面助手中，然后：

```
按资产表流程，处理这份剧本，第17集到最后
```

支持格式：PDF / Word / Excel
